# My Web

这是一个基于 React、TypeScript、Vite 和 Tailwind CSS 的落地页项目。

## 快速开始

```bash
npm install
npm run dev
```

默认开发地址：`http://localhost:5173`

## 常用命令

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
