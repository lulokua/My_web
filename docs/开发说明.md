# 开发说明

## 项目简介

本项目使用以下技术栈进行开发：

- **Next.js**（React 全栈框架，使用 App Router）
- **TypeScript**（类型安全的 JavaScript 超集）
- **Tailwind CSS**（原子化 CSS 框架）

## 环境要求

- Node.js 18.18 或更高版本
- npm / pnpm / yarn 任选其一（以下示例使用 npm）

## 安装依赖

在项目根目录执行：

```bash
npm install
```

## 常用命令

| 命令 | 说明 |
| --- | --- |
| `npm run dev` | 启动本地开发服务器（默认 http://localhost:3000） |
| `npm run build` | 构建生产环境产物 |
| `npm run start` | 运行生产环境构建结果 |
| `npm run lint` | 运行 ESLint 检查（如已配置） |

## 目录结构

```
My_web/
├── src/
│   └── app/
│       ├── layout.tsx     # 根布局
│       ├── page.tsx       # 首页（当前为 Hello World）
│       └── globals.css    # 全局样式（包含 Tailwind 指令）
├── docs/
│   └── 开发说明.md        # 本文档
├── public/                # 静态资源（按需创建）
├── next.config.ts         # Next.js 配置
├── tailwind.config.ts     # Tailwind CSS 配置
├── postcss.config.mjs     # PostCSS 配置
├── tsconfig.json          # TypeScript 配置
└── package.json
```

## 路径别名

`tsconfig.json` 中已配置 `@/*` 指向 `src/*`，例如：

```ts
import Foo from "@/components/Foo";
```

## 开发流程

1. 在 `src/app/` 下添加页面或路由（App Router 约定式路由）。
2. 公共组件建议放在 `src/components/`。
3. 使用 Tailwind 工具类编写样式，必要时在 `globals.css` 中扩展。

---

> 本项目基于 **Next.js + TypeScript + Tailwind CSS** 框架开发。
